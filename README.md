# 📊 Vrinda Store Sales Analysis Dashboard

## Project Overview

This project analyzes annual sales data of Vrinda Store using Microsoft Excel. The objective was to identify customer purchasing behavior, top-performing states, sales channels, and key business opportunities through an interactive dashboard.

---

## Tools & Techniques Used

- Microsoft Excel
- Data Cleaning
- Data Preprocessing
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Business Analysis

---

## Dashboard Features

- Orders vs Sales Analysis
- Sales Distribution by Gender
- Order Status Analysis
- Top 5 States by Sales
- Age Group vs Gender Analysis
- Channel-wise Sales Analysis
- Interactive Filtering using Slicers

---

## Key Business Insights

- Women customers contributed approximately **65%** of total sales.
- **Maharashtra, Karnataka, and Uttar Pradesh** were the top-performing states.
- Customers aged **30–49 years** generated nearly **50%** of total sales.
- **Amazon, Flipkart, and Myntra** were the leading sales channels.
- More than **90%** of orders were successfully delivered.

---

## Business Recommendation

To increase sales, Vrinda Store should focus marketing campaigns on **women customers aged 30–49 years** living in **Maharashtra, Karnataka, and Uttar Pradesh** by offering targeted discounts, promotions, and personalized campaigns through **Amazon, Flipkart, and Myntra**.
